Futebol das Galaxias
====================



Regra do Negócio e Modelagem

Ao iniciar o aplicativo, deverá ser criado uma nova Partida, lendo as Configurações padrão do Banco de Dados, na instalação já deve ter preparado uma configuração default.

Configuracoes:
id;
quantidadeGol;
tempoDaPartida;

Partida contém:
id;
quantidadeGolTimeA;
quantidadeGolTimeB;



Nas Configurações, o usuário poderá determinar o tempo máximo da partida, e a quantidade de gols para vencer a partida, a cada nova partida será lido da configurações para gerar no novo jogo.


Botão Novo Jogo, criará uma nova partida sem registros e irá zerar o cronometro.
Botão começar: inicia o cronometro e o jogo
Botão Pausar: pausa o tempo.


Botão Gol, abre o spinner para marcar assistencia e quem marcou o gol 

Botão Lance: mostra quem fez as assitencias e gols  da partida.
